# Changelog

本文件记录 DeepGraph 对使用者可感知的变化,风格参考
[Keep a Changelog](https://keepachangelog.com/)。

**维护约定:** 按 PR 标题聚合草稿(`gh release create <tag> --generate-notes`
生成初稿),攒到用户能感知出差别时再整理成一版、打一个 tag 发布。纯工程内部
重构、或用户无感的性能优化可以不单列。每版按 🆕 新增 / ⚡ 改进 / 🐛 修复
三类、用使用者看得懂的话写。

## v0.2 — 2026-06-10

### 看板 (dashboard)

- 🆕 全新知识图谱:领域 taxonomy 讲故事视图 + 实体关系网络图,可缩放/拖拽 (#37, #19)
- 🆕 Evidence 对比矩阵改为热力图,单元格按数值深浅上色 (#38)
- 🆕 中英双语界面 + 术语统一 + 可折叠面板 (#31)
- 🆕 9 个首页指标加了悬停说明 (#33, #36)
- ⚡ 首屏打开更快(指标缓存)(#34, #35)
- ⚡ 整页加载与交互更流畅,大数据下不再卡 (#38)
- 🐛 修复打开页面后会卡死/冻住 (#36)
- 🐛 修复页面变慢、数据库连接挂死的两处后端泄漏 (#33)
