# Changelog

本文件记录 DeepGraph 对使用者可感知的变化,风格参考
[Keep a Changelog](https://keepachangelog.com/)。

**维护约定:** 按 PR 标题聚合草稿(`gh release create <tag> --generate-notes`
生成初稿),攒到用户能感知出差别时再整理成一版、打一个 tag 发布。纯工程内部
重构、或用户无感的性能优化可以不单列。每版按 🆕 新增 / ⚡ 改进 / 🐛 修复
三类、用使用者看得懂的话写。

## v0.3 — 2026-06-11

### 研究方向接入 (agenda)

- 🆕 研究方向 YAML 一键接入:一句话方向 + 可选关键词/约束 → 自动落库为 agenda,并生成"系统理解回显"供提交者确认 (#41)
- 🆕 服务器收件夹:把 YAML 放进 `research_agendas/inbox/` 即自动解析入库,失败文件隔离并附原因 (#41)
- 🆕 多个 agenda 可同时活跃,各自绑定提交者;产出按 agenda 隔离互不可见 (#41)
- 🆕 idea 生成/信号采集/候选圈定按 agenda 关键词限定到相关子图,不同方向互不串题 (#41)
- 🆕 按 agenda 的 token 用量记账 + `/api/token_usage` 汇总(默认只统计不封顶;封顶可按 agenda 配置,超限干净暂停、可恢复)(#41)
- 🐛 修复存量 Postgres 库缺记账表导致 `/api/token_usage` 500 (#43)
