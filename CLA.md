## How To Sign

To sign this CLA for your GitHub account:

1. Read this file in full.
2. Comment on your pull request with the exact sentence below:

   `I have read the CLA.md and agree to the terms.`

3. Ask a maintainer to add your GitHub username to `cla-signers.json`.

Once your username is added to `cla-signers.json` on the default branch, the CLA check will pass automatically on your pull request.
